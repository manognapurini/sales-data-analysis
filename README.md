# Sales Data Analysis using Python

## Objective
To analyze sales data and extract meaningful insights using Python.

## Tools Used
- Python
- Pandas
- Matplotlib
- Google Colab

## Analysis Performed
- Data cleaning and formatting
- Monthly sales trend analysis
- Category-wise sales analysis
- Identification of top-performing products

## Key Insights
- Total sales amount to approximately 872,363
- Monthly sales show clear trends and seasonal variations
- A small number of products contribute significantly to total revenue

## Outcome
This project demonstrates practical data analysis and visualization skills using Python.
